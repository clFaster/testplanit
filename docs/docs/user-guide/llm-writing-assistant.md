# AI Writing Assistant

The rich text editor includes a built-in AI Writing Assistant that helps improve content directly within any rich text field (descriptions, steps, preconditions, etc.). This feature is available wherever the TipTap editor is used, provided the project has an active LLM integration.

## Accessing the AI Writing Assistant

1. Place your cursor in any rich text field or select text you want to improve
2. Click the **wand icon** in the editor toolbar
3. Choose an AI action from the dropdown menu

## Available Actions

**Writing Improvements:**

- **Improve Writing** — Improves grammar, clarity, and flow while maintaining the original meaning
- **Make Shorter** — Makes text more concise while keeping key information
- **Make Longer** — Expands text with more detail and context
- **Fix Grammar** — Corrects grammatical errors

**Change Tone:**

- **Professional** — Rewrites text in a professional tone
- **Casual** — Rewrites text in a casual, friendly tone
- **Formal** — Rewrites text in a formal tone

**Translate:**

- **English** — Translates text to English
- **Spanish** — Translates text to Spanish
- **French** — Translates text to French

**Content Enhancement:**

- **Summarize** — Summarizes text in a clear, concise way
- **Add Examples** — Adds relevant examples or use cases

## Using the Writing Assistant

1. The selected text (or full editor content) is sent to your configured LLM provider
2. The AI generates an improved version of the text
3. A side-by-side comparison shows the original text and the AI suggestion
4. Click **Accept Suggestion** to replace your text, or dismiss to keep the original

## Requirements

- The project must have at least one active LLM integration
- User must have access to the project
